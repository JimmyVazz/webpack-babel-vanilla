# Babel Webpack Starter

Ejemplo del uso de webpack y babel

### Version
1.1.0

## Uso

### Instalación

Instalar las dependencias

```sh
$ npm install
```

### Servidor de desarollo
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build para producción
Compilar y construir bundle

```sh
$ npm run build
```
